## 条件语句，循环语句

```js

// 字符串 比较的都是 asc码
var a = "a" > "b";

var a = "10" > "8"; // false  => var a = "1" > "8";


var a = NaN == NaN; // false

```


## 逻辑运算符

```
&& 与
|| 或
!  非
```


```js

// undefined, null, NaN, "", 0, false  => 布尔值 都是 false

var a = 1 && 2; // 2


var b = !123  // false  将值转为布尔值再取反

```
