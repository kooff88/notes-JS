hyperText markup language!!!  
超文本标记语言

```html
lang = "en" : 告诉搜索引擎，我们的网站是关于什么内容的。en: 英文的
<html lang="en">
  <head>
    <!-- utf-8 万国码 -->
    <meta charset="utf-8" />
  </head>
  <body>
    <!-- 中划线 -->
    <del>$50</del> 
  
  
  </body>

</html>


```
